# AI Companion · 陪读小精灵

一个给小朋友的**语音陪读小精灵**：按住说话，它用温柔的声音陪你读书、答疑、聊天；
一块**护眼水墨屏**显示对话与表情。硬件用 **ESP32-S3**，重活（识别/大模型/合成）全在云端。

> `ai_lamp`（智能学习台灯）的姊妹项目。这里聚焦"纯语音陪伴 + 极简屏"，砍掉摄像头与坐姿。

## 定位（范围已收敛）
- ✅ 语音陪读 / 聊天：按住说话 → 云 ASR → 大模型 → 云 TTS → 扬声器
- ✅ 极简显示：水墨屏（护眼、省电）显示对话文字 + 几个精灵表情状态
- ❌ 不做摄像头 / 拍照批改 / 坐姿（去掉最大的画质风险）
- ❌ 不做台灯外形

## 架构：ESP32 只当"瘦客户端"，云端做重活
```
┌────────── ESP32-S3（本地·瘦客户端）──────────┐        ┌──── 火山云（按需）────┐
│  麦克风(I2S) ─录音─▶                          │        │                      │
│  按键(按住说话) ──▶  联网转发 ──HTTPS──────────┼───────▶│  ASR 识别            │
│  水墨屏  ◀─文字/表情─                          │        │  豆包 LLM 陪聊        │
│  喇叭(I2S) ◀─播放─                            │◀───────┤  TTS 合成(要PCM直播)  │
└──────────────────────────────────────────────┘        └──────────────────────┘
```
思想同 `ai_lamp`：**本地廉价常开 + 云端偶发点火**（类比 VAD 门控云端识别）。

## 硬件（暂定 BOM）
| 部件 | 选型 | 备注 |
|------|------|------|
| 主控 | ESP32-S3（带 PSRAM）| I2S 进+出 + 屏幕都够 |
| 麦克风 | INMP441（I2S）| 数字麦，便宜好用 |
| 功放 | MAX98357A（I2S）+ 4/8Ω 小喇叭 | 数字功放 |
| 屏幕 | 水墨屏 Waveshare 2.9"/4.2"（SPI, SSD1680 支持局部刷新）| 护眼、省电；表情做成几张位图 |
| 输入 | 按键 ×1（按住说话）| 后续可加 |

> 屏幕取舍：水墨屏**擅长静态文字 + 偶尔换表情**，**不擅长流畅动画**。想要"活泼会动的桌宠"可换 GC9A01 圆形 IPS；这里选水墨屏主打**护眼安静**。

## 开发路线（先 Pi 后 ESP32）
1. **先在 Pi/PC 上用 Python 把语音闭环调定型**：按住说话录音 → ASR → 豆包 → TTS(PCM) → 播放，调好提示词与交互节奏。
2. 交互定稿后，把这套"瘦客户端"**移植到 ESP32-S3**（原型可 MicroPython，产品转 ESP-IDF/C）。
3. ESP32 面对的始终是**已验证的确定目标**，不边调算法边啃固件。

## 关键技术选择
- **TTS 要 PCM/WAV，不要 MP3**：ESP32 免解码，直接喂 I2S 播放。
- **先用"按住说话"(push-to-talk)**：录完再传（submit/query，~2s 出字）；成熟后再上流式 ASR(wss)。
- **密钥走环境变量/.env**，绝不入库（沿用 ai_lamp 的 env 约定）。
